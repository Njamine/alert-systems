# Forest Dashboard

Live dashboard for forest disturbance alerts with:

- Flask API (`/api/alerts`, `/api/summary`)
- PostgreSQL/PostGIS backend
- Leaflet interactive map + filters + table

## Project path

`D:\alert sytem web\forest-dashboard`

## Structure

- `backend/` - Flask API, DB config, helpers
- `frontend/` - HTML template
- `static/` - CSS and JavaScript served by Flask
- `instructions/` - step-by-step setup, run, and GitHub deployment guides

## Quick start

1. Read `instructions/01_setup_local.md`
2. Read `instructions/02_run_and_test.md`
3. Read `instructions/03_publish_to_github.md`
